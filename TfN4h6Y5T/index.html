15. 忽略文件

# git在管理文件时，忽略某些文件
# .gitignore文件

15.1 使用

# 项目目录下创建.gitignore文件
# 写入要忽略的文件名(也可以是.gitignore文件)
*.py # 表示忽略所有的以.py结尾的文件
files/ # 忽略files目录下的所有文件
!a.py # 忽略以.py结尾的文件但排除a.py
*.py[c|a|d] # 忽略.pyc .pya .pyd结尾的文件
